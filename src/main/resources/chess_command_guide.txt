=== Chess Command Guide ===

1. 이동 명령어:
   - 사용법: move <시작좌표> <목표좌표>
   - 설명: 체스 기물을 지정된 위치로 이동합니다.
     예시: move a2 a3

2. 이동 가능 경로 명령어:
   - 사용법: locations <좌표>
   - 설명: 특정 좌표에 있는 기물이 이동할 수 있는 모든 경로를 표시합니다.
     예시: locations a2

3. 캐슬링 명령어:
   - 사용법: castling <킹 좌표> <룩 좌표>
   - 설명: 킹 기물과 룩 기물을 캐슬링 합니다.
     예시: castling e1 h1

4. 종료 명령어:
   - 사용법: exit
   - 설명: 체스 게임을 종료합니다.

=== 명령어를 입력하여 게임을 시작하세요! ===
